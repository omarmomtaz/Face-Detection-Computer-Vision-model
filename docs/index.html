04 The Code

Clean API.
Zero friction
integration.

Drop FaceDetector into any existing Python codebase. Composable flags, structured return data, upgrade-safe architecture.

  • 4 lines to full detection with eye + smile analysis
  • Structured dict output — bbox, eyes[], smile bool
  • All parameters are optional with sane defaults
  • Upgrade path: swap cascade → DNN without API changes
  • pip install and run — zero configuration required
main.py Python 3.x 
# ─── Core Integration ─────────────────────
from main import FaceDetector
import cv2

detector = FaceDetector()
image    = cv2.imread('photo.jpg')

# ─── Detect with all layers ───────────────
faces = detector.detect_faces(
  image,
  detect_eyes  = True,
  detect_smile = True
)

# ─── Structured return data ───────────────
for face in faces:
  x, y, w, h = face['bbox']
  eyes  = face['eyes']    # list of (x,y,w,h)
  smile = face['smile']   # bool | None

# ─── Render output ────────────────────────
result = detector.draw_detections(
  image, faces, draw_eyes=True
)

cv2.imwrite('output.jpg', result)
